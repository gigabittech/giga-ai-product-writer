Generate unique, high-converting, and SEO-optimized WooCommerce product descriptions using Anthropic Claude AI.

== Description ==

Giga AI Product Writer is the ultimate companion for WooCommerce store owners who want to scale their content without sacrificing quality. Powered by the state-of-the-art Claude 3.5 Sonnet model, this plugin understands your products like a human copywriter.

= Features =
* **State-of-the-art AI:** Uses Claude-3.5-Sonnet for superior reasoning and creative writing.
* **WooCommerce Native:** Deeply integrated with product attributes, categories, and tags.
* **SEO Optimized:** Generates Meta Titles and Descriptions compatible with Yoast SEO and Rank Math.
* **Quality Scoring:** Real-time analysis of readability, SEO keyword density, and benefit-to-feature ratio.
* **Brand Voice (Pro):** Train the AI on your existing descriptions to match your unique brand tone perfectly.
* **Bulk Generation (Pro):** Update hundreds of products in the background with a single click.

= External API Disclosure =
This plugin connects to the Anthropic Claude API to generate product descriptions.
Your product data (title, attributes, price, category) is sent to Claude for processing.
No customer personal data is transmitted. 
API Terms: https://www.anthropic.com/legal/usage-policy

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/giga-ai-product-writer` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the **Giga AI -> Settings** screen to configure your Claude API key.

== Frequently Asked Questions ==

= Does it work with Yoast SEO? =
Yes! It automatically detects Yoast and Rank Math and writes meta fields directly to them.

= How many products can I generate for free? =
The free version allows for 5 generations per month. Upgrade to Pro for unlimited use.
